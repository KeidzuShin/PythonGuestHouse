# Python Web Based Project
**Overview**

A web-based system designed to implement booking marketing display, management, user authentication, and reporting. The system streamlines reservation handling and provides clear transaction records for administrative and operational use.

Make sure to include you own .env file for the admin authentication.

example can be found in the file named: ```.env.example.txt```

To run the main dashboard: ```python app.py```

**Tech Stack**

Make sure you have these
- Backend: Python, Flask
- Database: MySQL (SQLAlchemy)
- Frontend: Python, HTML, CSS
- Authentication: Session-based login system
- Reporting: PDF generation for transaction reports

**Features**

- Marketing display page for guest house promotion and information
- Booking management with Create Read Update Delete (CRUD) functionality
- User authentication and access control
- Transaction recording and management
- PDF report generation
- Database integration for persistent data storage
